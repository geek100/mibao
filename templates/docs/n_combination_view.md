N组合视图(适用于原生活动区域展示)
-----
###### 1.概述
```text
1.app开发时经常会遇到某一块视图变动很频繁;
2.如首页某一区域需要做活动;
3.在满足以上1和2的同时需要改变视图的呈现方式;

计划:
   第一阶段每个占位用图片来呈现;
   第二阶段可在每个占位上增加一些修饰视图;
```

###### 2.呈现形式
```java
 --------------------------------           ------------------------------------
|              |      |          |         |               |                    |
|              |      |          |         |               |                    |

|                  |             |         |---------------|                    |
|                  |             |         |               |                    |
|                  |             |         |               |--------------------|
|                  |             |         |               |                    |      ......
 --------------------------------           ------------------------------------
```
